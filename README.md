# Audible Insights - Book Recommendation System

## Features
- NLP-based recommendations
- Clustering & similarity models
- Streamlit UI

## Run Steps
1. Preprocess data:
   python src/data_preprocessing.py

2. Feature engineering:
   python src/feature_engineering.py

3. Train model:
   python src/model.py

4. Run app:
   streamlit run app/str_app.py
